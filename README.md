#  Java Mini Projects

This repository contains multiple beginner-friendly Java projects built using core concepts of Java and OOP.

---

##  Projects Included

### 1. Calculator
A simple calculator that performs basic arithmetic operations.

**Features:**
- Addition, Subtraction, Multiplication, Division
- Exception handling (divide by zero)

**Concepts Used:**
- Scanner
- switch-case
- try-catch

---

### 2. Student Result System
Calculates total marks, percentage, and grade.

**Features:**
- Input student name and marks
- Calculates total and percentage
- Displays grade (A/B/C)

**Concepts Used:**
- Scanner
- if-else
- basic calculations

---

### 3. Number Guessing Game
A fun game where the user guesses a randomly generated number.

**Features:**
- Random number generation
- User input
- Hints (Too High / Too Low)

**Concepts Used:**
- Random class
- while loop
- conditions

---

### 4. Simple Interest Calculator
Calculates simple interest based on user input.

**Features:**
- Input principal, rate, and time
- Calculates simple interest

**Concepts Used:**
- Scanner
- arithmetic logic

---

### 5. To-Do List App
A console-based task manager.

**Features:**
- Add tasks
- View tasks
- Remove tasks
- Menu-driven system

**Concepts Used:**
- ArrayList
- loops
- conditional statements

---

##  Concepts Covered

- Java Basics
- OOP Concepts
- Exception Handling
- Collections (ArrayList)
- User Input Handling
